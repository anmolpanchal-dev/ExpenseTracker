# Expense Tracker

A simple web-based Expense Tracker application to help you manage and visualize your daily expenses. Built with HTML, CSS, and JavaScript.

## Features
- Add, edit, and delete expenses
- View a list of recent expenses
- Visualize spending with charts
- Persistent storage using localStorage
- Responsive and user-friendly interface

## Project Structure
```
index.html           # Main HTML file
README.md            # Project documentation
assets/images/       # Image assets (icons, logos, etc.)
css/main.css         # Stylesheet
js/app.js            # Main application logic
js/chart.js          # Chart rendering logic
js/recentExpenses.js # Recent expenses component
js/store.js          # Local storage management
```

## Getting Started
1. Clone or download this repository.
2. Open `index.html` in your web browser.
3. Start tracking your expenses!

## Usage
- **Add Expense:** Enter the details and click 'Add'.
- **View Expenses:** See your recent expenses listed on the main page.
- **Visualize:** Check the chart for a graphical view of your spending.
- **Delete/Edit:** Remove or update expenses as needed.

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
